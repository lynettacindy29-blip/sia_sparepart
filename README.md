**Sistem Informasi Akuntansi (SIA) Sparepart**



Sistem ini dirancang untuk mengintegrasikan pencatatan stok barang (sparepart) dengan pelaporan keuangan secara otomatis. Fokus utama pengembangan adalah pada **internal control** dan **integritas data**.



* Fitur Utama

  1. **Manajemen Inventaris:** Pencatatan stok masuk dan keluar secara real-time.
  2. **Otomatisasi Akuntansi:** Posting otomatis dari transaksi ke Jurnal Umum dan Buku Besar.
  3. **Laporan Keuangan:** Menghasilkan Laporan Laba Rugi dan Neraca secara instan.
  4. **Validasi Data:** Menerapkan kontrol input untuk mencegah data ganda atau salah input (Audit Trail).



* Teknologi yang Digunakan

  1. **Bahasa Pemrograman:** PHP Native
  2. **Database:** MySQL
  3. **Server:** Laragon / XAMPP



* Cara Penggunaan

  1. Import file `db\_sia\_sparepart.sql` ke database Anda.
  2. Sesuaikan konfigurasi database di folder `config/`.
  3. Akses melalui browser di `localhost/sia\_sparepart`.
